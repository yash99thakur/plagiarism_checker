Abstract

With the internet advancing day by day and the sources of information increasing, the whole learning process has been revolutionized with ease of access and unlimited opportunities. But the silver-lining sooner or later fades away and the downside to this process is faced. The students have started blindly copying from the internet just in order to get their assignments completed, hence giving rise to copy education rather than an inspired education. This hampers not only with the learning process but when such individuals are presented to the industry scale work their productivity, creativity decreases.

Aim of Project

The main objective of this project is to develop a solution to detect plagiarism in documented text file and also in source codes of the programs. We do not wish to undermine someone’s capabilities and their morale by simply judging them on the basis of the work they submit and making the over conscious. The simple objective is to make them use their own creativity and enhance their problem-solving capabilities. Taking inspirations from someone’s works is good but blindly copying them poses greater threats

Explaination to Working of project

In our solution we have provided the user option to give the original source in text format or either we can scrape the web using google API so that we can get the source from the which data have been copied .After getting the the plag text and information of orignal text we try to apply a machine learning model which takes out the similarities between the two texts based on cosine similirity principle. in which we make set of unique words using the both text and find the similarity between them in vector space. After that our python application fetches the response from the model and display it to user in ha human readable format providing the source 's URL if applicable

Unique features

  • We have used language processing model which dosent depend on string matching but it uses the root words to find similarity in vector space. so it automatically neglects spelling mistaked which is generally done to hide the plagrism from string matching models of pag checkers

Cosine Similarity

Cosine similarity is a metric used to measure how similar the documents are irrespective of their size. Mathematically, it measures the cosine of the angle between two vectors projected in a multi-dimensional space. The cosine similarity is advantageous because even if the two similar documents are far apart by the Euclidean distance (due to the size of the document), chances are they may still be oriented closer together. The smaller the angle, higher the cosine similarity.
